Methodology

Portfolio Value (BTC) = Futures wallet balance + Unrealized PNL, converted to BTC.

Robot P&L = Current portfolio (BTC) − External deposits (BTC). Isolates bot performance from capital injections.

ROI (BTC) = Robot P&L / External deposits. Denominated in BTC, not USD.

Monthly PNL = Realized PNL + Funding Fees + Commissions per calendar month, each converted to BTC at historical daily price.

Forecast = Average monthly PNL ± 1σ projected forward. Optimistic = avg + σ, Pessimistic = avg − σ.

Trend = Linear regression on monthly PNL values. Positive slope = improving performance.
